# Checklist: Revisão de Copy — EMB

> Responsável: @estrategista-emb (revisão) + @copywriter-emb (autor)
> Score mínimo: 80% para aprovação

## Fundamentos (6 itens)
- [ ] Big Idea presente e clara?
- [ ] Mecanismo Único mencionado?
- [ ] Promessa principal no headline/abertura?
- [ ] Copy fala com UMA pessoa (você)?
- [ ] Tom de voz do expert (não genérico)?
- [ ] Linguagem do avatar (palavras que ELE usa)?

## Estrutura (5 itens)
- [ ] Hook prende atenção nos primeiros 3 segundos?
- [ ] Transição lógica entre seções?
- [ ] CTA claro e específico?
- [ ] CTA repetido em pontos estratégicos?
- [ ] Hierarquia visual facilita leitura rápida?

## Persuasão (6 itens)
- [ ] Dor antes do prazer?
- [ ] Prova social presente?
- [ ] Autoridade demonstrada (não apenas afirmada)?
- [ ] Especificidade (números concretos)?
- [ ] Urgência/escassez REAL?
- [ ] Garantia que remove risco?

## Avatar & Empatia (4 itens)
- [ ] Dor principal articulada com precisão?
- [ ] Desejo profundo conectado?
- [ ] Objeções principais endereçadas?
- [ ] Leitor se identifica?

## Anti-IA (4 itens)
- [ ] Zero expressões da blacklist?
- [ ] Tom natural e humano?
- [ ] Variação de tamanho de frase?
- [ ] Sem estruturas repetitivas?

## Score: ___/25 = ___%

```
>= 80% → Entregar
60-79% → Entregar com flag dos itens a corrigir
< 60%  → Refazer antes de entregar
```
