# Universo Algorítmico — Como Furar Bolha de Verdade

O tema define o universo. O universo define quem vê. Se o tema é "5ABS", o algoritmo entrega pra quem consome fisioterapia clínica. Se o tema é "vácuo", o algoritmo entrega pra quem consome fitness. Mesmo expert. Audiência 100x maior.

Este arquivo cobre a lógica de universos algorítmicos e como usá-la para gerar temas que realmente furam bolha — não apenas temas com hooks melhores dentro do mesmo universo pequeno.

---

## O Problema: Teto Algorítmico

O algoritmo do Instagram (e TikTok, YouTube Shorts) não entrega conteúdo baseado em "quem segue quem". Entrega baseado em **universo temático** — o conjunto de assuntos que uma pessoa consome regularmente.

Quando todo conteúdo de um expert fala sobre o mesmo nicho específico, o algoritmo categoriza aquele perfil dentro de um universo pequeno. E entrega pra quem? Pras mesmas pessoas que já consomem esse nicho. Não importa se o hook é melhor ou pior — o conteúdo está operando dentro do mesmo teto.

O sintoma: views baixos, não fura bolha, mesma audiência, crescimento estagnado.

A causa: o conteúdo compete num universo algorítmico pequeno demais.

**Exemplo concreto:**
- Expert de reestruturação abdominal publica sobre "lógica evolutiva", "5ABS", "diástase" → algoritmo cruza com quem consome fisioterapia pélvica, reabilitação, Pilates clínico → pool de ~5K pessoas
- Mesmo expert publica sobre "vácuo" (trend fitness), "29 músculos do abdômen", "o que cesárea faz no corpo" → algoritmo cruza com quem consome fitness, treino, maternidade, corpo humano → pool de 500K+ pessoas

O expert é o mesmo. A expertise é a mesma. O que muda é o **universo em que o tema compete**.

---

## A Regra Central

**O tema precisa ser interessante para um LEIGO CURIOSO. A profundidade revela que é feito por um EXPERT.**

Se o tema SÓ interessa a quem já está no nicho do expert, não é conteúdo de atração. É conteúdo de profundidade (que tem sua função — mas não fura bolha).

O conteúdo de atração compete nos universos AMPLOS onde o avatar do expert também vive — mas como pessoa, não como profissional/cliente.

---

## As 3 Camadas Dentro de Cada Reel

Cada reel de atração faz o trabalho completo em 3 camadas, dentro do mesmo conteúdo:

### Camada 1 — Atração Ampla (Hook + Aterrissagem — primeiros 5-7s)

O ponto de entrada pertence ao universo AMPLO: fitness, corpo humano, treino, saúde, maternidade, estética, carreira, superação. O algoritmo distribui para esse universo.

**O hook NÃO menciona:**
- O nome do método/produto do expert
- Termos que só o nicho entende
- Nada que filtre para "só profissionais" ou "só clientes do nicho"

**O hook MENCIONA:**
- Algo que qualquer pessoa curiosa sobre corpo/saúde/fitness/carreira pararia pra ver
- Dado, inversão, paradoxo, provocação que funciona no universo amplo

### Camada 2 — Profundidade (Body — 10-90s)

O corpo do reel demonstra expertise de outro nível. Mecanismos, dados, explicações que vão além do óbvio. O leigo aprende algo fascinante. O profissional reconhece: "essa pessoa sabe mais do que eu".

**O que acontece naturalmente:**
- Leigo: "isso eu não sabia → vou compartilhar"
- Profissional: "essa pessoa entende num nível que eu não sabia que existia → vou seguir"

A profundidade é o filtro natural. Não precisa dizer "esse conteúdo é pra profissionais". A competência demonstrada faz o trabalho.

### Camada 3 — Posicionamento (Ponte + CTA — últimos 10-15s)

O fechamento. Não precisa ser venda direta. A demonstração de competência já fez o trabalho de atrair. O posicionamento pode ser assinatura, CTA suave, provocação que gera debate, ou ponte pela ameaça ao negócio (ver `fechamento-cta-assinatura.md`).

**O funil natural:**
1. Leigo curioso para o scroll → "isso eu não sabia"
2. Profissional reconhece expertise → segue, salva, compartilha
3. Lead qualificado navega perfil → descobre a especialização → entra no funil

## Como Identificar Universos para Qualquer Expert

### Passo 1: Mapear os universos que o AVATAR consome (não o expert)

Perguntar: "Quando o avatar do expert NÃO está em modo profissional/cliente, o que ele assiste no Instagram/TikTok?"

| Tipo de expert | Avatar consome (fora do nicho) |
|---|---|
| Fisioterapeuta/Ed. Físico | Fitness, treino, bodybuilding, corpo, saúde, maternidade |
| Nutricionista | Receitas, saúde, bem-estar, estética, longevidade |
| Advogado empresarial | Negócios, empreendedorismo, economia, política |
| Psicólogo | Comportamento, relacionamentos, autoconhecimento, neurociência |
| Dentista | Estética, autoestima, beleza, saúde |
| Consultor de marketing | Negócios, tech, criatividade, cultura pop, redes sociais |

### Passo 2: Encontrar a INTERSEÇÃO

A interseção é onde o expert tem propriedade real E o universo é amplo.

**Fórmula:** Expertise do expert ∩ Universo amplo que o avatar consome = Temas de atração

Exemplo para fisioterapeuta de reestruturação abdominal:
- Expertise: abdômen, core, respiração, diástase, pós-parto, avaliação postural, fisiculturismo
- Universos amplos do avatar: fitness, treino, bodybuilding, corpo, maternidade, estética

**Interseções:**
- Vácuo (expertise real + universo fitness)
- Abdômen de fisiculturista (expertise real + universo bodybuilding)
- Pós-cesárea (expertise real + universo maternidade)
- Respiração e abdômen (expertise real + universo fitness/saúde)
- Core real vs. core de academia (expertise real + universo treino)

### Passo 3: Validar propriedade

O tema precisa passar neste teste: "O expert tem experiência, caso real, dado ou história pessoal sobre esse tema que outra pessoa não teria?"

Se sim → tema com propriedade → prosseguir.
Se não → tema genérico → descartar (qualquer influenciador poderia fazer, e a profundidade que faz o profissional reconhecer expertise não vai existir).

## O Modelo Dra. Liliana (Referência)

A Dra. Liliana é a referência prática para experts de saúde que aplicam esse modelo:

**O que ela FAZ:**
- Temas universais de saúde feminina (anticoncepcional, hormônios, produtos de farmácia) — universo de milhões
- Profundidade médica que nenhum influenciador fitness consegue replicar
- Nomes dramáticos para mecanismos ("Menopausa Química") — big ideas recontáveis
- Nunca diz "esse conteúdo é pra médicas"

**O que ACONTECE:**
- Leiga assiste, aprende, compartilha → alcance viral
- Médica assiste, reconhece profundidade → segue como autoridade
- O perfil cresce no universo amplo (saúde feminina) → e a especialização médica vem como consequência

**O padrão replicável:** Não fazer conteúdo "para [profissão] que quer [resultado profissional]". Fazer conteúdo SOBRE [tema amplo] que fascina qualquer pessoa — e a profundidade técnica faz o profissional se identificar sozinho.

## Distribuição de Conteúdo: Atração vs. Profundidade

Nem todo conteúdo precisa ser de atração ampla. O expert precisa de ambos:

| Tipo | Função | % sugerido | Universo |
|---|---|---|---|
| **Atração** | Furar bolha, trazer seguidores novos | 40-60% | Amplo |
| **Profundidade** | Demonstrar domínio do nicho, converter | 20-30% | Nicho |
| **Conexão** | Criar relacionamento, mostrar bastidor | 10-20% | Pessoal |
| **Conversão** | Vender, gerar ação | 10% | Nicho + oferta |

O erro comum: 100% profundidade (expert fala só de nicho → teto baixo). Quando o perfil precisa crescer, inverter para 50%+ atração.

A métrica que valida: **% de contas alcançadas que NÃO seguiam antes**. Se esse número sobe nos reels de atração vs. profundidade, a estratégia de universo está funcionando.

## Formatos que Funcionam para Atração Ampla

### React a Conteúdo Viral
Expert reage a vídeo popular do universo amplo com perspectiva técnica que ninguém tem. O vídeo original já tem distribuição no universo amplo — o react herda parte desse universo.

### Demonstração Visual no Corpo/Prática
Expert mostra algo no próprio corpo ou na prática que é visualmente impactante. Visual para o scroll > texto. E demonstração é prova implícita de competência.

### Dado/Fato que Fascina Qualquer Pessoa
Números contra-intuitivos são compartilháveis. O compartilhamento é o principal vetor de viralização no Instagram.

### Inversão de Crença Universal
Algo que TODO MUNDO faz e está errado — não só o nicho. A base de pessoas que acredita (e portanto para pra questionar) é muito maior que inversão de crença profissional.

## Checklist de Universo Algorítmico (Aplicar a Todo Tema de Atração)

- [ ] **Este tema compete em qual universo?** Se a resposta é "nicho do expert" → não é atração.
- [ ] **Um leigo curioso assistiria?** Se precisa ser profissional pra se interessar → não é atração.
- [ ] **O expert tem propriedade real?** Se qualquer influenciador poderia fazer → não diferencia.
- [ ] **O hook evita termos de nicho?** Se tem jargão ou nome de método no hook → trocar.
- [ ] **As 3 camadas estão presentes?** Hook amplo → profundidade que demonstra expertise → posicionamento.
