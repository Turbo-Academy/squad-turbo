# Visão Geral dos Funis — Referência do Estrategista Turbo

## Mapa de Decisão

```
QUAL É O OBJETIVO PRINCIPAL?
│
├── CAPTURAR LEADS → Isca Gratuita / Quiz / Low Ticket
├── VENDER AGORA → Low Ticket / Campanha Rápida / Venda Interna
├── LANÇAR UM PRODUTO → PLF Pago / PLF Gratuito / Seed / Meteórico
├── FAZER UM EVENTO → Webinar / Workshop / Desafio / Imersão / Presencial
├── VENDER ALTO TICKET → Aplicação / Sessão Estratégica / Sala Secreta
└── VENDER SERVIÇO → Funil de Serviços (Freelancer / Agência)
```

## Perguntas de Diagnóstico (Triage do Estrategista)

Para escolher o funil certo, responder:

1. **O expert tem audiência?**
   - Sim (>1k seguidores ativos) → PLF, Webinar, Desafio
   - Não / pequena → Seed Launch, Meteórico, PLF Gratuito

2. **Qual o ticket-alvo?**
   - Até R$197 → Low Ticket, Tripwire
   - R$200–2K → Seed, Meteórico, Workshop, Desafio
   - R$500–5K → PLF, Webinar
   - R$3K+ → Aplicação, Sessão, High Ticket

3. **Qual a urgência?**
   - Precisa de resultado em até 7 dias → Meteórico, Campanha Rápida
   - 1–2 semanas → Webinar, Workshop, Lançamento Pago Semanal
   - 4–8 semanas → PLF Clássico
   - Contínuo → Perpétuo, Low Ticket, Isca

4. **Tem produto pronto?**
   - Não → Seed Launch (vende antes de criar)
   - Sim → Qualquer outro

5. **Qual o objetivo de negócio?**
   - Construir lista → Isca Gratuita, Quiz
   - Primeira venda → Low Ticket, Tripwire
   - Receita recorrente → Lançamento Pago Semanal, Perpétuo
   - Venda única grande → PLF, High Ticket

## Sequências de Funil Recomendadas

**Expert novo (sem audiência, sem produto validado):**
`Isca Gratuita → Seed Launch → PLF Gratuito → PLF Pago`

**Expert com audiência pequena:**
`Low Ticket → Webinar → PLF Pago`

**Agência/serviço:**
`Isca Gratuita → Sessão Estratégica → Serviços`

**Receita recorrente rápida:**
`Lançamento Pago Semanal (5+1) → Perpétuo`

**High ticket:**
`Isca Gratuita → Webinar → Aplicação → Sala Secreta`
