# Value Ladder — Framework Russell Brunson

> Baseado em "DotCom Secrets" de Russell Brunson

## Conceito

A Value Ladder é a progressão de ofertas que leva o cliente de um primeiro contato gratuito até o produto mais caro e personalizado. Cada degrau entrega mais valor e cobra mais.

## Estrutura da Value Ladder

```
                                    ┌─────────────┐
                                    │  PREMIUM     │ R$10K-100K+
                                    │  (1:1)       │ Mentoria/Done-for-you
                                ┌───┴─────────────┴───┐
                                │  HIGH TICKET        │ R$2K-10K
                                │  (Grupo/Programa)   │ Curso + comunidade
                            ┌───┴─────────────────────┴───┐
                            │  MID TICKET                 │ R$200-2K
                            │  (Curso/Workshop)           │ Produto principal
                        ┌───┴─────────────────────────────┴───┐
                        │  LOW TICKET / TRIPWIRE              │ R$7-197
                        │  (Ebook/Mini-curso)                 │ Primeira compra
                    ┌───┴─────────────────────────────────────┴───┐
                    │  ISCA GRATUITA (Lead Magnet)                │ Grátis
                    │  (PDF, Checklist, Aula, Quiz)               │ Captura email
                    └─────────────────────────────────────────────┘
```

## Regras da Value Ladder

1. **Cada degrau resolve UM problema** — não tudo de uma vez
2. **O gratuito gera desejo pelo pago** — mostra o QUÊ, não o COMO
3. **Cada compra qualifica para a próxima** — quem compra R$47, tem mais chance de comprar R$497
4. **Preço sobe = personalização sobe** — mais caro = mais acesso ao expert
5. **Nunca pule degraus** — construa confiança progressivamente
